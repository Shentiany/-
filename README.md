## 📣 简介
Python 是一门富有表达力的语言。此项目使用Python实现了一个简易的爬虫，爬取了国家级大学生创新创业训练计划平台的项目名称，并绘制了简单的图表。
********

## 🔰 词云
使用Python第三方库jieba（中文分词）和wordcloud分别切分项目名称和制作词云。
